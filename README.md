# 🎬 YouTube Dashboard (UI Clone)

A static front-end clone of the YouTube dashboard built using **HTML and CSS**.
This project focuses on replicating the layout and design of YouTube’s interface.

---

## 🔗 Live Demo

`https://youtube-dashboard-clone.netlify.app/`

---

## 📌 Features

* 🎥 YouTube-like homepage layout
* 📂 Sidebar navigation (Home, Subscriptions, Library, etc.)
* 🔍 Search bar UI
* 🎞️ Video grid with thumbnails
* 📱 Responsive design (if implemented)
* 🎨 Clean and modern UI styling

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/SrujanRaj-cse/Youtube-Dashboard.git
```

2. Navigate to the project folder:

```
cd Youtube_DashBoard/Youtube2.0
```

3. Run a local server:

```
python3 -m http.server 8000
```

4. Open in browser:

```
http://localhost:8000/html/dashboard.html
```
